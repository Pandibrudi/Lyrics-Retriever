# A simple lyrics collector

This Python project collects all lyrics from a specific artist from genius using the genius API

# Usage

Download the code or clone the repository via `git clone git@github.com:Pandibrudi/Advanced-Python-Project.git`

Ideally use in own virtual environment. Install the [requirements](requirements.txt) via `pip install -r requirements.txt`

You will need to make a genius [developer/api account](https://genius.com/developers) and create a client access token. Save this in a `.env` in the project folder.

Use the terminal to change into the directory of the scripts via `cd /your/path`

Start the program via `python main.py "Artist Name"` - make sure to write the artist name in quotes!
